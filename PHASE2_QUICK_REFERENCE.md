6. Approved â†’ moves to Scheduled

## ðŸ”„ Common Workflows

### Workflow 1: Plan Week's Content
1. Open **Content Calendar**
2. Click future dates to add events
3. Create 7 posts (one per day)
4. Assign content types (mix posts, stories, reels)
5. Set all to "Scheduled" status
6. View color-coded week overview

### Workflow 2: Generate & Schedule from AI Ideas
1. Open **AI Ideas Generator**
2. Generate ideas (5 per click)
3. Copy idea you like
4. Go to **Content Calendar**
5. Create event and paste copied content
6. Set scheduled time from AI recommendation

### Workflow 3: Review Performance & Create Similar Content
1. Open **Post Rankings**
2. Sort by Engagement Rate (click column header)
3. Note top performing posts
4. Identify patterns (hashtags, content type, time)
5. Go to **AI Ideas Generator**
6. Generate ideas in same category
7. Schedule in **Content Calendar** at similar times

### Workflow 4: Bulk Update Status
1. Open **Bulk Management**
2. Filter by "Draft"
3. Select all (or specific items)
4. Click "Bulk Actions" â†’ "Change Status"
5. Enter "pending_approval"
6. Go to **Approval Workflow** to review

### Workflow 5: Team Approval Process
1. Team member creates content in **Calendar**
2. Sets status to "Pending Approval"
3. Manager opens **Approval Workflow**
4. Filters by workflow stage
5. Reviews content, adds notes
6. Approves or rejects
7. Approved content auto-moves to Scheduled
8. Rejected content goes back to Draft for revision

## â“ FAQs

**Q: How do I reschedule a post?**
A: Click the event on the calendar, change the scheduled date, and save.

**Q: Can I edit content after approval?**
A: Yes, but status changes back to draft. Re-submit for approval.

**Q: How many AI ideas can I generate?**
A: Unlimited. Each click generates 5 new ideas.

**Q: What happens to rejected content?**
A: Status changes to "Draft" so you can edit and resubmit.

**Q: Can I bulk delete drafts?**
A: Yes. Filter by "Draft", select items, click Bulk Actions â†’ Delete.

**Q: Do calendar events sync with Instagram?**
A: Not currently. This is a planning tool. Publishing requires manual action or future API integration.

**Q: Where do Post Rankings get data?**
A: From media_insights table populated by Instagram API fetches.

**Q: How accurate are AI posting time recommendations?**
A: Based on general best practices. Refine with your account's analytics.

**Q: Can multiple team members approve content?**
A: Yes, workflow supports multi-stage reviews with different approvers.

**Q: Is there a content limit?**
A: No hard limit. Database scales with your needs.
